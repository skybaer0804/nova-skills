# my-skills

Personal Claude Code skills for Next.js frontend development.

## Skills

- `nextjs-accessibility-review` — WCAG 2.1 AA compliance review for components and pages
- `nextjs-component-design` — Component API and architecture design before implementation
- `nextjs-performance-review` — Performance audit: rendering strategy, bundle, images, re-renders
- `nextjs-state-design` — State type and location decisions (server/client/URL/local)
- `nextjs-design-token-consistency` — Design token and Tailwind consistency review after styling

claude plugins install skybaer0804/my-skills
